# Translator_App

Translator_App jest to aplikacja napisana przy użyciu języka Python.  
Umożliwia ona sprawne tłumaczenie słów z języka na język (z dostępnych w biblotece).  
Zapisuje wszystkie wpisania do bazy danych i umożliwia użytkownikowi poźniejszy dostęp do nich.  
Aplikacja raportuje również wszystko co dzieje się podczas wpisywania do bazy danych.  
Aplikacja również pozwala na prostą naukę fiszek z wpisanych słówek.
